# Contributing to VividAI

Thank you for your interest in contributing to VividAI! We welcome contributions from the community.

## Code of Conduct

Please note that this project is released with a [Contributor Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Create a feature branch (`git checkout -b feature/amazing-feature`)
4. Make your changes
5. Commit your changes (`git commit -m 'Add some amazing feature'`)
6. Push to the branch (`git push origin feature/amazing-feature`)
7. Open a Pull Request

## Development Setup

```bash
# Install dependencies
npm install

# Load the extension in Chrome
# 1. Open chrome://extensions
# 2. Enable Developer mode
# 3. Click "Load unpacked"
# 4. Select the repository directory
```

## Pull Request Guidelines

- Keep PRs focused and small
- Include clear description of changes
- Reference related issues
- Add tests if applicable
- Update documentation as needed

## Reporting Bugs

- Use the GitHub Issues tracker
- Provide detailed description of the bug
- Include steps to reproduce
- Specify your browser version and OS

## Suggesting Enhancements

- Use the GitHub Issues tracker
- Provide detailed use cases
- Explain expected behavior
- Include any relevant screenshots or examples
